# PlayStation Interview Prep

This folder is for the Sony Interactive Entertainment video codec / cloud gaming role.

The resume and HR screening are already cleared, so this prep package focuses on the 45-minute hiring-manager interview and the later technical panel.

## Files

- `01_HM_45_Min_Runbook.md`: main guide for the Seema Kataria / Jason Wang hiring-manager interview.
- `02_Story_Bank.md`: fill-in scripts for technical and behavioral stories.
- `03_Technical_Study_Sheet.md`: codec, GPU, concurrency, and profiling notes.
- `04_Mock_Questions.md`: practice questions and answer outlines.
- `05_Questions_To_Ask_HM.md`: strong questions to ask the hiring managers.
- `06_Company_And_Person_Research.md`: public research notes and role signals.
- `07_Prep_Schedule.md`: focused timeline for the HM screen and panel.

## Core Interview Message

The main message to repeat across answers:

> I am strongest in C++ systems, graphics/GPU programming, debugging, and performance analysis. This role is a strong match because video codec and cloud gaming work needs low-level engineering judgment across CPU, GPU, memory, timing, and validation. I can own features end to end and ramp quickly on codec-specific details.
