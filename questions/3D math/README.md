# 3D Math Practice Questions

This folder contains the standalone 3D graphics math practice questions from the original root `questions/` folder. Duplicate individual CoderPad 3D math questions were removed; `questions/coderpad_rendering/CoderPad_All.cpp` is kept as the combined CoderPad practice file.

## Included Topics

- Vector operations, dot/cross products, length, normalization, angles, reflection, projection, and distance
- Matrix construction, transform composition, point/direction transforms, LookAt, camera basis, and world-to-screen conversion
- Ray-plane, ray-sphere, ray-AABB, frustum-sphere, billboard, interpolation, clamp, and rectangle math
