# NeetCode 150

This folder contains the generated NeetCode 150 roadmap practice files.

Each problem has two files:

- `Prompt_NC###_Problem_Name.cpp`: CoderPad-style prompt starter.
- `Answer_NC###_Problem_Name.cpp`: answer workspace starter.

Every file includes a description, examples with explanations, an `Implement:` section, a problem-specific `Solve...` stub, `RunTests()`, and `RUN_TESTS()`.

See `NEETCODE_150.md` for the complete topic-ordered index.
