# LeetCode Practice Questions

This folder contains LeetCode-style C++ practice problems.

## NeetCode 150

The generated NeetCode 150 roadmap files are in `neetcode_150/`.

- `neetcode_150/Prompt_NC###_Problem_Name.cpp`: CoderPad-style prompt starter.
- `neetcode_150/Answer_NC###_Problem_Name.cpp`: answer workspace starter.
- `neetcode_150/NEETCODE_150.md`: complete topic-ordered index.

## Extra Non-Roadmap Solved Examples

These older `Q##_...` files remain directly in this folder because they are not duplicates of the generated NeetCode 150 roadmap files:

- `Answer_Q02_Minimum_Changes_To_Make_Alternating_Binary_String.cpp`
- `Answer_Q03_Basic_Calculator_II.cpp`
